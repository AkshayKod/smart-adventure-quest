## 🌟 About the Project

**Smart Adventure Quest** is a complete, interactive text-based adventure game built entirely in Python. Designed to demonstrate strong foundational programming logic, this project beautifully encapsulates key computer science concepts:

- **Condition Handling & Control Flow**
- **Iterative Loops (For/While)**
- **Modular Functions**
- **Data Structures (Lists, Dictionaries, Sets)**
- **Exception Handling (`try-except`)**
- **File Handling (I/O)**

Whether you are a beginner looking for an inspiring project or a student preparing for an internship assessment, this game acts as the perfect capstone portfolio piece.

## 🚀 Key Features

| Feature | Description |

## 🎮 How to Play

### 1. Prerequisites
Ensure you have Python 3.x installed on your system. Download it from [python.org](https://www.python.org/).

### 2. Run the Game
Clone the repository and run the script in your favorite terminal:

```bash
git clone https://github.com/yourusername/smart-adventure-quest.git
cd smart-adventure-quest
python smart_adventure_quest.py
```

### 3. Survival Guide
- **Health:** You start with `100 HP`. Wrong answers, traps, or monster attacks will decrease it. At `0 HP`, it's Game Over!
- **Inventory:** Certain choices yield items (e.g., `Magic Scroll`, `Sword`). Having these changes your options in future levels.
- **Score:** Maximize your points to achieve Legendary ranks!
